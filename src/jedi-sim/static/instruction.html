errorCodes.json Field Description

Complete guide to understanding error code structure and field definitions

Field Name Type Description
Z0 int Generator state when error occurs (e.g., 0x06 indicates error state)
Z1 int Simplified error code (e.g., 30, 90, etc.)
Z2 int Display bitmap (bit 0-7, usually 0 indicates all normal)
Z3_Phase int Phase when error occurs (high 4 bits)
Z3_Class int Error category (low 4 bits)
Z4Z5_ErrorCode int Generator error code (little-endian format, e.g., 804, 134)
Z6Z7_ErrorData int Data related to error (e.g., temperature, etc., e.g., 5)
Description string Error description information
